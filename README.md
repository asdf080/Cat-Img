# Cat Img Maker

### 🔎소개

### 📅제작 기간
2023.12.30

### 🗂개발 환경
- HTML
- CSS
- JavaScript
- React

### 🎈주요 기능
- 오픈 API를 이용한 무작위 고양이 이미지 생성 기능
- 로컬 스토리지를 이용한 이미지 저장 기능
