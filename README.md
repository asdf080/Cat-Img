# Cat Img Maker

### 🔗링크
https://asdf080.github.io/Cat-Img/

### 🔎소개
텍스트를 입력하면 랜덤한 고양이 사진에 텍스트가 입력되는 토이 프로젝트입니다.

### 📅제작 기간
2023.12.30

### 🗂개발 환경
- HTML
- CSS
- JavaScript
- React

### 🎈주요 기능
- 오픈 API를 이용한 무작위 고양이 이미지 생성 기능
- 원하는 텍스트가 사진에 입력되는 기능
- 로컬 스토리지를 이용한 이미지 저장 기능
